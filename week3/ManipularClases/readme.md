# Manipular Clases
Usando la Solución llamada PrimerProyecto, diseñado en la semana la práctica formativa de la semana 2, cuyo entorno hace referencia a:

> En un instituto terciario deben registrar el nombre, apellido y documento de los futuros postulantes para luego asignarles el curso y procesar la inscripción …

Se pide resolver cada uno de los siguientes puntos:
- Agregar al diseño un control comboBox para indicar el tipo de documento del postulante. Los ítems deben ser DNI, Pasaporte, Extranjero.
- Crear la clase Postulante cuyos atributos son los del siguiente diseño: Nombre, Apellido, Tipo, Documento.
- Codificar en el botón INGRESAR la carga de datos en la clase. Para verificar si el ingreso es correcto mostrar en un cuadro de diálogo el contenido.
> Los contenidos de los controles TextBox y ComboBox se almacena en la propiedad text.

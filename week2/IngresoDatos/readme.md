# Ingreso de datos
1. Ingresar los datos desde los textBox y mostrarlos en cuadro de dialogo. Usar el botón “Aceptar”.

2. Tomar el punto 1, pero los datos deben ser ingresados también en un cuadro de diálogo.

3. Usando el botón “Cálculo” solicitar con un cuadro de diálogo el ingreso de dos números (es un cuadro de dialogo para cada dato) y mostrar cuál es el número más grande.

> La forma de declaración, lectura de datos numéricos y uso de condicional lo viste en la semana 1 y lo ejercitaste desde el modo consola. Simplemente debes adaptarlo a este nuevo modo.
